🌌 AstroExplore
A space discovery website to explore planets and learn amazing facts about our universe.
🚀 Features
Interactive Planet Cards: Engaging visuals and details for every planet.
📊 Quick Facts Table: Fast access to key planetary data at a glance.
📱 Fully Responsive Design: Optimized for seamless viewing on desktop, tablet, and mobile.
🌟 Smooth Animations: Polished UI transitions for a modern feel.
📧 Newsletter Signup: Easily subscribe for cosmic updates.
🛠️ Technologies
HTML5 & CSS3: Core structure and advanced styling.
JavaScript: Interactive logic and dynamic elements.
Font Awesome: Modern vector icons.
NASA Images: High-quality, authentic space imagery.
📥 Quick Start
Download: Clone or download all project files to your local machine.
Folder Structure: Ensure you keep the folder structure intact so CSS, JS, and image assets load correctly.
Launch: Simply open index.html in any modern web browser to start exploring.
